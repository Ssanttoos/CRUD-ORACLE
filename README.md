Um projeto CRUD conectado ao banco de dados, utilizei o oracle por ter um pouco mais de familiaridade. É um sistema bem simples que consiste em criar uma tabela com o nome, email e numero de telefone, podendo editar as informações e deletar tudo de uma forma visual.
Utilizei o forms pra deixar de uma forma mais visual e interativa.
(de inicio era para ser algo mais academico pegando informações e alunos, porém acabei mudando de ideia)
